Provides a way to save annotation to annotation file.

Function opens and loads all annotation details from annotation file into memory. After that it creates a new annotation and adds annotation details to it, or it updates previous annotation if the image was previously annotated in this annotation file.

Annotation file which is being edited is passed as 1st parameter to this function.

If for some reason that file is not available function throws FileNotFoundError.

Parameters
jsonFilePathQString Path to a annotation file in which annotation is going to be stored to.
imageFilePathQString Path to a image for which annotation is being made.
classNameQString Name of a class which is used for annotating when drawing a shape.
coordinatesLinkedList of all points in a annotation shape, in format x,y (std::pair<int,int>).
